## 📊 Model Performance

| Metric | Score |
|---|---|
| Accuracy | 92.85% |

## 🛠 Tech Stack
Python · Pandas · Scikit-learn · NLTK · TF-IDF · Logistic Regression · Streamlit

## 🚀 How to Run
pip3 install -r requirements.txt
python3 -m streamlit run app.py
